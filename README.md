# 🏦 Indian Bank Capstone Project

## 📌 Overview

This project focuses on analyzing banking data to extract meaningful insights using SQL and data visualization tools. It simulates real-world banking operations such as transactions, customer behavior, and account management.

## 🎯 Objectives

* Analyze transaction patterns
* Identify customer behavior trends
* Build insightful dashboards
* Optimize data queries using SQL

## 🛠️ Tech Stack

* SQL (MySQL)
* Power BI / Tableau
* Excel (for preprocessing)

## 🗂️ Database Schema

* **AccountMaster**
* **CustomerMaster**
* **BranchMaster**
* **TxnMaster**

## 📊 Key Features

* Complex SQL queries (joins, aggregations, subqueries)
* Transaction analysis (credit/debit trends)
* Customer segmentation
* Dashboard visualization
